# ChatBotTipsKehidupan
Chatbot ini berfokus pada tips berekehidupan sehari-hari dengan menggunakan NLTK sehingga tulisan kita bisa didapat sesuatu yang unik dan chabot akan merepons dengan balasan nya yang sudah dipasangankan di dalam variable intens
